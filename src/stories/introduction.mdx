<h1>What is React Seat Toolkit?</h1>

<span>You're looking at an <b>Open Source</b> and completely free to use <b>NPM library</b> which requires as less as 5 lines of code to get started with.</span> 

<br/>

<br/>

<span>It is a <b>high performance, JSON based tool</b> which allows you to <b>build and render complex seat maps right within your web application</b> without the need for any external services. You control both the data and the design. </span>

<br/>

<br/>

<span>There are two modes in which you can use React Seat Toolkit:</span>

<ul>
  <li><b>Designer Mode:</b> This mode allows you to design your seat map where you can add, remove, edit and move seats around. You can also change the seat colors, add labels bind them into categories and sections and much much more.</li>
  <li><b>User Mode:</b> This mode allows you to view the seat map. You can interact with the seat map by selecting seats. There are handlers which can be provided to control what happens when a seat is selected, deselected, hovered over etc.</li>
</ul>

<hr/>

<br/>

<span>Here are a few things you can build with React Seat Toolkit:</span>

<ul>
  <li><b>Conference Hall Seat Maps</b></li>
  <li><b>Stadium Seat Maps</b></li>
  <li><b>Movie Theater Seat Maps</b></li>
  <li><b>Bus Seat Maps</b></li>
  <li><b>Flight Seat Maps</b></li>
  <li><b>And anything else that you can think of...</b></li>
</ul>

<br/>

<h2 className="border-b-0 font-semibold">What is React Seat Toolkit built with?</h2>

<span>React Seat Toolkit is built with React, D3.js, Redux and TailwindCSS and Storybook.</span>
